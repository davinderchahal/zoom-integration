
# Zoom Integration

Integrating Zoom Meetings into CodeIgniter: A Developer's Guide



## Configuration
1. Download the code
2. Create a new database in mysql and import zoom-meeting.sql file.
3. Add database credential in database.php file in application/config folder.
4. See the Zoom_lib file in application/libraries folder and configure the zoom credentials.

That's it. Enjoy the zoom meeting
